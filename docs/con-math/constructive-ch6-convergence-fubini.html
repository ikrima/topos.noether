Convergence Theorems & Fubini

When can we exchange limits and integrals? Monotone and dominated convergence give the answer. Fubini's theorem handles iterated integrals.

The fundamental question of integration theory: when does

lim I(fₙ) = I(lim fₙ)?

Not always! We need conditions. Three notions of convergence matter here:

Definition 8.1:

• In measure: ∀ε, ∃N: |fₙ - f| < ε on "most" of any integrable set
• Almost everywhere: For each integrable set A, |fₙ - f| < ε eventually on most of A
• Almost uniformly: |fₙ - f| < ε uniformly on most of A

Monotone Convergence Theorem

Increasing sequences with bounded integrals converge

Term n
5

Theorem 8.6

Let (fₙ) be an increasing sequence of integrable functions. Then (fₙ) converges in measure to an integrable function f if and only if lim I(fₙ) exists. In that case, (fₙ) converges almost everywhere, and I(f) = lim I(fₙ).

Dominated Convergence Theorem

Bounded by an integrable function? Then limits commute.

Theorem 8.8 (Lebesgue Dominated Convergence)

Let (fₙ) be a sequence of integrable functions converging in measure to an integrable function f, and let g be an integrable function such that |fₙ| ≤ g for all n.

Then: lim I(|f - fₙ|) = 0

Product Integrals & Fubini's Theorem

Iterated integration equals direct integration on the product

Theorem 9.7 (Fubini)

I(J(f)) = (I × J)(f) = J(I(f))

Integrate in either order — same result!

The convergence theorems are the workhorses of integration theory:

The Pattern:

Monotone: fₙ ↑ and sup I(fₙ) < ∞
   ⟹ limit exists and I commutes with lim

Dominated: |fₙ| ≤ g integrable
   ⟹ I(|f - fₙ|) → 0

Fubini: f integrable on X × Y
   ⟹ integrate in any order

These are the tools that make measure theory work.
