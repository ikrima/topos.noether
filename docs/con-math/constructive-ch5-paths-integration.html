Paths & Cauchy's Theorem

For analytic functions, the integral around any closed path is zero. The most powerful theorem in complex analysis.

A path γ: [a, b] → ℂ is a continuous, piecewise differentiable curve in the complex plane.

Definition 3.1:

γ(t) = x(t) + i·y(t)    (t ∈ [a, b])

γ is piecewise differentiable: γ' exists except at finitely many points

The line integral of f along γ is:

∫_γ f(z) dz = ∫_a^b f(γ(t)) · γ'(t) dt

Cauchy's Integral Theorem

For analytic functions, closed path integrals vanish

∮_γ f(z) dz = 0

For any analytic f on a simply connected region U, and any closed path γ in U

Simply Connected Regions

Regions with no holes allow Cauchy's theorem to work

○

Simply Connected

Every closed path can be continuously shrunk to a point.
Examples: Disk, half-plane, ℂ itself

◎

Not Simply Connected

Some paths cannot shrink without leaving the region.
Examples: Punctured disk ℂ - {0}, annulus

Cauchy's theorem has remarkable consequences:

Corollary 3.15:
If γ₁ and γ₂ have the same endpoints,
then ∫_{γ₁} f dz = ∫_{γ₂} f dz

The integral depends only on endpoints, not the path!

This is path independence — the complex analogue of the fundamental theorem of calculus.
