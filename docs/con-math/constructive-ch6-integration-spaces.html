Sections 1 & 2: Integration Spaces

The Daniell Integral

Start with simple functions, extend by completion. The constructive path to integration avoids measure theory's set-theoretic machinery.

Classical integration requires measure theory — a foundation built on non-constructive set operations.

The Daniell approach is different: start with an integral on "simple" functions, then extend it to all integrable functions via explicit approximations.

The Setup:

• X = a set with inequality ≠
• L = partial functions X → ℝ
• I: L → ℝ = the integral

The triple (X, L, I) must satisfy four axioms...

The Four Axioms

Definition 1.1: What makes (X, L, I) an integration space

Axiom 1.1.1

Linearity & Lattice Closure

f, g ∈ L, α, β ∈ ℝ ⟹ αf + βg, |f|, f∧1 ∈ L

I(αf + βg) = αI(f) + βI(g)

L is closed under linear combinations, absolute value, and truncation. I is linear.

Axiom 1.1.2

Series Convergence

Σ I(fₙ) < I(f) ⟹ ∃x: Σ fₙ(x) < f(x)

If the series of integrals converges to less than I(f), there's a witnessing point.

Axiom 1.1.3

Non-Triviality

∃p ∈ L: I(p) = 1

There exists at least one function with non-zero integral.

Axiom 1.1.4

Limit Conditions

lim I(f∧n) = I(f)    and    lim I(|f|∧n⁻¹) = 0

Truncations from above and below behave correctly.

Full Sets

The constructive "almost everywhere"

Definition 2.4

A subset F ⊆ X is full if there exists an integrable function f with dmn(f) ⊆ F.

Full sets are "big" — they contain the domain of some integrable function.

The key to extending I is the Lebesgue Series Theorem:

Theorem 2.15:

If Σ I(|fₙ|) converges, then:

• F = {x: Σ|fₙ(x)| converges} is a full set
• f = Σfₙ with domain F is integrable
• lim ‖f - Σₖ₌₁ⁿ fₖ‖₁ = 0

This gives L¹ its structure: the complete extension is itself complete as a metric space under ‖f‖₁ = I(|f|).
