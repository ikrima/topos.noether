Section 4: Compactness

Total Boundedness & Compactness

Compact = Complete + Totally Bounded. This constructive definition makes infinite sets manageable.

A space is totally bounded if for every ε > 0, it can be covered by finitely many balls of radius ε:

Definition 4.1: An ε-approximation to X is a finite set Y ⊂ X
such that for each x ∈ X, there exists y ∈ Y with ρ(x, y) < ε.

X is totally bounded if such a Y exists for every ε > 0.

This is stronger than boundedness. A bounded set has finite diameter; a totally bounded set has finite covers at every scale.

Compact (Definition 4.7)

A metric space that is both complete and totally bounded. This makes analysis tractable: infinite sets become finitely approximable.

Why Not Classical Definitions?

The usual definitions are non-constructive

✗ Non-Constructive

Open Cover Definition

"Every open cover has a finite subcover"

Problem: There's no routine method to extract finite subcovers from arbitrary open covers.

✗ Non-Constructive

Sequential Definition

"Every sequence has a convergent subsequence"

Problem: Selecting the subsequence requires unbounded search; no algorithm guaranteed to work.

✓ Constructive

Complete + Totally Bounded

"Complete and has finite ε-approximations for all ε > 0"

Why it works: Both conditions are computationally meaningful. We can construct the approximations and verify convergence.

Compactness has powerful consequences:

Corollary 4.3: If f: X → ℝ is uniformly continuous
on a totally bounded space X, then sup f and inf f exist.

On a compact space, continuous functions achieve their bounds. The supremum and infimum are not just theoretical—they're computable.

This is why compact spaces are so important: they turn infinite problems into finite ones.
