Section 2: Finite-Dimensional Spaces

Finite Dimensions & Best Approximation

Finite-dimensional spaces are locally compact. Best approximation requires "at most one closest point" — a constructive condition.

A normed space is finite-dimensional if it has a finite basis — vectors e₁, ..., eₙ such that every x can be written uniquely as

x = u¹(x)e₁ + u²(x)e₂ + ... + uⁿ(x)eₙ

The coordinate functionals u¹, ..., uⁿ are bounded linear functionals. The dimension n is an invariant of the space.

The Fundamental Dichotomy

Finite-dimensional ⟺ locally compact

Theorem 2.6

A normed linear space is locally compact if and only if it is finite-dimensional.

Locally Compact ⟺ Finite-Dimensional

Every finite-dimensional normed space is a Banach space — automatically complete and separable.

Best Approximation

Finding the closest point in a subspace

Theorem 2.12

If F is a finite-dimensional subspace of a normed space X over ℝ, and a ∈ X has at most one closest point in F, then a has a unique closest point in F.

"At most one" is the constructive condition that makes best approximation work.

The classical theorem says: every point has a closest point in a finite-dimensional subspace. But constructively, we need more:

Definition 2.11: a has at most one closest point in F if:

For all x, x' ∈ F with x ≠ x', there exists y ∈ F with
max{ρ(a,x), ρ(a,x')} > ρ(a,y)

This condition ensures that if two points look equally close, we can find a closer one — preventing indecision.

Quotient Spaces

Collapsing a subspace to zero

Definition 2.9

Let Y be a located linear subset of X. The quotient space X/Y has seminorm:

‖x‖_{X/Y} = ρ(x, Y)

Two vectors x, x' are equal in X/Y if ‖x - x'‖_{X/Y} = 0, i.e., if ρ(x-x', Y) = 0.

Key Uses:

• Reducing dimension: dim(X/Y) = dim(X) - dim(Y)
• Induction proofs for best approximation
• Studying hyperplanes as quotients by ℝe
