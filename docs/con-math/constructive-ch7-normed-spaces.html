Norms & Bounded Linear Maps

A norm measures size. Bounded linear maps preserve structure. Banach spaces are complete — the arena for modern analysis.

A normed linear space combines two structures: vector space operations and a notion of distance.

Definition 1.2: A norm ‖·‖ satisfies

• ‖αx‖ = |α| ‖x‖   (scaling)
• ‖x + y‖ ≤ ‖x‖ + ‖y‖   (triangle)
• ‖x‖ = 0 ⟹ x = 0   (positive definite)

The norm induces a metric: ρ(x, y) = ‖x - y‖. Different norms give different geometries.

Bounded Linear Mappings

Five equivalent conditions for continuity

Proposition 1.5: Equivalent Conditions

  • (i) u is continuous
  • (ii) u is uniformly continuous
  • (iii) u is bounded on the unit sphere
  • (iv) u is bounded on each bounded set
  • (v) ∃c > 0 : ‖u(x)‖ ≤ c‖x‖ for all x
Key Insight:

For linear maps, local behavior determines global behavior.
Bounded on the unit sphere ⟹ bounded everywhere.

Hyperplanes & Linear Functionals

The kernel of a bounded linear functional is a hyperplane

Proposition 1.8

If u is a nonzero bounded linear functional, then ker u is a hyperplane.

Conversely, every hyperplane is the kernel of some bounded linear functional.

Here's a constructive subtlety: a bounded linear functional may not have a norm.

Definition 1.9: u is normable if

‖u‖ = sup{|u(x)| : ‖x‖ ≤ 1}

exists as a real number.

The key theorem: u is normable ⟺ ker u is located.

Locatedness — the ability to compute distances — is the bridge between boundedness and normability.
