SU(3) → Lift → Cocycle

The function SU(3) → S⁵, sending a matrix to its last column, is a principal Spin(3)-bundle. Such a thing is described by a nontrivial function from the equator S⁴ ⊂ S⁵ to Spin(3), and we can then project this into 3-dimensional space.

What you see here is (part of the data of) a lift of the function to a cocycle valued in a richer algebraic structure, namely a model for the categorical group String(3). Such a cocycle contains more information than is contained here, but each loop shown is a pair of functions from S⁴ minus the north and south poles to the space of paths in Spin(3) that start at the origin (again projecting to 3-dimensional space), and both ending at the same point.

For the experts: this is the two object components of the unique nontrivial Ω̂Spin(3) → PSpin(3)-valued cocycle on the 5-sphere, for a specific globular hypercover.
