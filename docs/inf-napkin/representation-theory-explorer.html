What is Representation Theory?

Representation theory is the art of understanding abstract algebraic structures by seeing them as matrices acting on vector spaces. Instead of thinking about a group G abstractly, we represent each element g as a matrix ρ(g), turning group multiplication into matrix multiplication.

Core Idea: A representation (V, ρ) of a group G is a homomorphism ρ: G → GL(V), assigning to each group element an invertible linear transformation. The character χV(g) = Tr(ρ(g)) captures the essential information.

The fundamental theorem (Maschke) says that for finite groups over ℂ, every representation decomposes into irreducible pieces. The character table encodes all this information in a compact square matrix.

Character Tables

The character table is a square matrix where rows are irreducible representations and columns are conjugacy classes. It encodes all the essential information about a group's representations.

Key Properties

  • Square: # of irreps = # of conjugacy classes
  • Sum of squares: Σ (dim Vi)² = |G|
  • Row orthogonality: ⟨χV, χW⟩ = δVW
  • Column orthogonality: Σ χ(c₁)χ(c₂)* = |G|/|c₁| · δc₁c₂

Orthogonality of Characters

Characters of irreducible representations form an orthonormal basis with respect to the inner product ⟨χ, ψ⟩ = (1/|G|) Σ χ(g)ψ(g)*.

Maschke's Theorem

For a finite group G and field k with char(k) ∤ |G|, the group algebra k[G] is semisimple. This means every representation is completely reducible — it decomposes as a direct sum of irreducibles.
