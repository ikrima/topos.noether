Biomimetic Reservoir Intelligence

A renderer and a visual system face the same problem: the world is too large to resample from scratch, so each maintains a small transported summary of what is likely to matter next.

The eye does not reservoir-sample pixels. It reservoir-samples explanations of pixels.
active proposals transported memory residual novelty bounded resampling
01 · Common skeleton

Not a camera. A bounded belief.

The tempting analogy is that a saccade is like moving a camera. The more useful analogy is that a saccade is a proposal operator. It chooses which evidence gets promoted into a small, weighted model of the world.

Renderer translation: do not ask only “which neighboring pixel can I reuse?” Ask “which existing hypothesis should survive after motion, task, evidence, and residual novelty have all competed?”
Rₜ = {(xᵢ, wᵢ, cᵢ, τᵢ)}ᵢ₌₁ᴷ xᵢ sample / light path / object hypothesis / surface patch wᵢ contribution or plausibility cᵢ confidence, precision, reuse validity τᵢ provenance, age, transform history Rₜ₊₁ = Resample(Reweight(Transport(Rₜ) ∪ Propose(yₜ₊₁)))

03 · Residual sampling

The useful signal is not the image. It is the mismatch.

Prediction moves a reservoir forward. Evidence arrives. The system should spend new samples on the part that the transported reservoir failed to explain.

μ⁻ₜ = (Tₜ)# μₜ₋₁ ρₜ = νₜ − μ⁻ₜ attention ∝ first-order residual + changing residual + singular novelty + curvature

In a renderer, that means less blind accumulation and more targeted refresh near disocclusions, hard visibility flips, specular sparkles, and regions where old samples are confident but wrong.

04 · Failure mode

Hallucination is stale reservoir governance.

A stale hypothesis becomes dangerous when confidence decays slower than evidence disagreement grows. In rendering this looks like ghosting, stuck lighting samples, stale shadows, or a denoiser hallucinating structure that evidence no longer supports.

The healthy response is not merely “turn off temporal reuse.” It is to keep confidence as a first-class quantity and force refresh only when confidence and mismatch become incompatible.

if confidence_high && evidence_mismatch_high: force_refresh() else: reuse_with_RN_gate()

05 · Back to ReSTIR / GRIS

Biology is useful only if it returns as an algorithm.

The safe formulation for graphics researchers is not “copy the brain.” It is: use biological vision as a generator of missing control signals for real-time sampling—confidence, residual novelty, adaptive capacity, object permanence, and active proposal selection.

Immediate upgrades

  • RN reuse gate: replace binary temporal validity with a density-ratio trust estimate.
  • Object-space reservoirs: reuse by object/material/motion group, not only screen neighborhood.
  • Foveated allocation: variable candidate count from residual, entropy, curvature, and gaze/task importance.
  • Hallucination control: track overconfident stale samples and force refresh on mismatch.

Falsifiable predictions

  • Curvature heatmaps should correlate with ghosting, shimmer, fireflies, and stale shadow artifacts.
  • Object-space reservoirs should outperform screen-space reuse under strong camera motion.
  • Entropy-aware adaptive capacity should reduce fireflies more efficiently than uniform extra samples.
  • RN-style gates should beat depth/normal rejection near disocclusions.
struct BioReservoir {
    Sample sample;
    float  weightSum;
    float  targetPdf;
    float  confidence;     // precision / reuse validity
    float  entropy;        // unresolved multimodality
    float  curvature;      // transport order disagreement
    float  novelty;        // singular residual proxy
    uint   age;
    uint   objectId;
    uint   materialId;
    uint   semanticGroup;
};

BioReservoir updatePixelReservoir(Pixel p) {
    BioReservoir prev = loadTemporalReservoir(p);
    Geometry g = loadGBuffer(p);
    Motion m = loadMotion(p);

    // 1. Predictive transport: move old belief forward.
    BioReservoir predicted = transportReservoir(prev, g, m);

    // 2. Compare transported measure to incoming evidence.
    float rn          = estimateDensityRatio(predicted, g);
    float singularity = estimateSingularNovelty(predicted, g);
    float curvature   = estimateTransportCommutator(p);

    predicted.confidence *= exp(-abs(log(max(rn, EPSILON))));
    predicted.confidence *= exp(-lambdaK * curvature);

    if (singularity > singularThreshold)
        predicted.confidence = 0.0;

    CandidateSet C;
    C.add(proposeTemporal(predicted));
    C.add(proposeSpatialNeighbors(p));
    C.add(proposeSameObject(g.objectId));
    C.add(proposeSameMaterial(g.materialId));
    C.add(proposeLightAndBSDF(p));
    C.add(proposeResidualNovelty(p));

    BioReservoir R = initReservoir();

    for (Candidate c : C) {
        float target   = evaluateTarget(c, g);
        float proposal = evaluateProposalPdf(c, g);
        float trust    = c.semanticTrust * exp(-lambdaK * c.curvature);
        float w        = target / max(proposal, EPSILON) * trust;
        updateWeightedReservoir(R, c.sample, w);
    }

    R.entropy    = estimateReservoirEntropy(R);
    R.curvature  = curvature;
    R.novelty    = singularity;
    R.confidence = saturate(R.confidence);

    int K = chooseReservoirCardinality(R.entropy, R.curvature, R.novelty, perceptualSalience(p));
    maybeAllocateAuxReservoirs(p, K);

    storeReservoir(p, R);
    return R;
}
06 · Production shape

The pass architecture is modest.

The theory sounds large, but the first implementation is a set of scalar control fields around an existing ReSTIR/GRIS pipeline.

Biomimetic ReSTIR pass map
diagnostic-first integration

The deeper object is not sampling.

It is maintenance of a bounded, transported, weighted empirical measure under partial observation.

Rendering: world → light transport measure → reservoir → image Vision: world → sensory measure → belief reservoir → perception AI: stream → latent hypothesis measure → attention reservoir → action

The radical version is coherent only because the engineering version is concrete: active proposal generation, weighted evidence integration, transported memory, bounded resampling, and residual-driven refresh.
