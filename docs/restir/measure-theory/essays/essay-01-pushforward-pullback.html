5. What the Jacobian is really correcting

If the transport map stretches a patch of path space, a fixed number of particles now covers a larger area; their density goes down. If it compresses the patch, their density goes up. GRIS-style correction is not merely an algebraic weight. It is the bookkeeping needed when transported probability mass changes density.

6. Curvature: when reuse order matters

Spatial reuse and temporal reuse are often tuned separately. But the hard cases are exactly where their order matters. Reuse spatially then temporally; now reuse temporally then spatially. If the results disagree, the local transport geometry is unstable.

Curvature diagnostic: compare temporal-after-spatial with spatial-after-temporal. The separation is a local path-dependence signal, useful for reducing reuse confidence and allocating fresh samples.

This gives a principled version of a familiar renderer instinct: near disocclusion, thin geometry, specular glints, or fast-changing visibility, trust history and neighbors less.

8. Algorithmic reading

The formulation suggests a renderer implementation that augments reuse with three diagnostics: compatibility from pullback scoring, density correction from transport, and instability from curvature/residual.

for pixel p:
    prior = empty_measure()
    attention = 0

    for source q in temporal_and_spatial_sources(p):
        T = build_transport(q, p)             // motion vector, path shift, object map...
        J = estimate_density_change(T)        // analytic or learned Jacobian

        for candidate xq in reservoir[q]:

            score = target[p](xp)              // pull current question back

9. Final compression

ReSTIR/GRIS can be read as a pushforward/pullback calculus over a screen-time bundle of path-space reservoirs. Reuse maps push empirical measures into the current pixelŌĆÖs path space. The current pixelŌĆÖs target, validity tests, and residuals pull back along those maps to score compatibility. Jacobians correct transported density. Curvature appears when spatial and temporal transports fail to commute, and the residual measure tells the renderer where transported belief stopped explaining current evidence.
