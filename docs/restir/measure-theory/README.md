# The Geometry of ReSTIR / GRIS

A static visual essay series on ReSTIR, GRIS, measure transport, reservoir geometry, residual novelty, and curvature-guided sampling.

## Quick start

Open:

```text
index.html
```

in a browser.

For stricter browser behavior, run a tiny local server from this directory:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Conceptual arc

1. **Pushforward & Pullback** — Samples move forward; questions move backward.
2. **Residual Measure Transport** — The renderer samples what prediction failed to explain.
3. **Biomimetic Reservoir Intelligence** — Vision and ReSTIR as bounded empirical-measure maintenance.
4. **The Shape of a Reservoir** — A reservoir is a compressed shape in path space.
5. **Curvature of Reuse** — Noncommuting reuse operators reveal instability.
6. **A Pixel Is Not a Point** — Pixels are base points with hidden path-space fibers.
7. **The Geometry of Light Transport** — Rendering equation, GRIS transport, Jacobians, residuals, and curvature unified.
8. **Curvature-Guided Sampling** — A practical adaptive renderer where variance becomes geometry.

## Notes

Each essay is a self-contained HTML artifact with embedded CSS and JavaScript. There are no external runtime dependencies.
