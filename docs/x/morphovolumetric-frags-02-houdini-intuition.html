§1

The Homomorphism

The Morphovolumetric Fragmentation Site begins with regions of medium equipped with surface/interior exchange laws [1]. SideFX Houdini's fluid solvers — FLIP, SPH, and its voxel-based pipeline — already implement this structure computationally.

The mapping is not a loose analogy. It is structurally precise: Houdini's simulation domain is a morphovolumetric fragment diagram, its solver steps are the bisheaf descent conditions, and its numerical errors are residuals in the exact sense of the framework.

Each voxel is a cell with assigned volume. Shared faces between adjacent voxels are internal interfaces that must cancel during reconstruction [1]. External faces form the boundary. The fragmentation identity holds exactly:

V(domain) = Σᵢ V(voxelᵢ) ← volume is additive A(∂domain) = Σᵢ A(∂voxelᵢ) − 2·Σ A(shared_face) ← boundary cancellation

In Houdini, this is not a theorem to prove — it's the data structure itself. The SOP/DOP pipeline encodes the fragment diagram.

§5

Vortex Cores as Gluing Obstructions

The framework proposes that "particle-like events appear when local fragment laws cannot glue, internal interfaces fail to cancel, or curvature jumps persist under refinement" [1]. In fluid dynamics, this description matches vortex cores with striking precision:

Gluing Obstruction Properties [1]

  • ●Local fragment laws cannot glue
  • ●Internal interfaces fail to cancel
  • ●Curvature jumps across interface
  • ●Flux leaks through boundary
  • ●Torsion frames fail cocycle
  • ●Defect persists under refinement

Vortex Core Properties

  • ●Velocity undefined at center (multi-valued)
  • ●Circulation ∮u·dl ≠ 0 (non-trivial holonomy)
  • ●Pressure singularity (curvature blowup)
  • ●Enstrophy concentration (flux leak)
  • ●Frame rotation around core (torsion)
  • ●Persists across all grid resolutions

§6

The Residual Tower = Adaptive Mesh

The framework defines a residual tower: as ε decreases, the sites approach exactness [1]:

⋯ → E_ε₂ → E_ε₁ → E₀ (ε₂ < ε₁)

In Houdini, this is adaptive mesh refinement — VDB's sparse hierarchical grid or Houdini's particle upresing. Each level adds cells where the current level's residual is too high. The tower is literally the sequence of simulation resolutions.
